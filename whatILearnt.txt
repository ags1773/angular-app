**    Routing - routerLink   **
*******************************

Adding CSS class to active element using routerLinkActive

routerLinkActive = 'activeClass':
- Used to add an active class to the active element
- can be placed on the element itself or its wrapper

here, we have placed the routerLinkActive on the wrapper of <a></a>
<li role="presentation" class="nav-item" routerLinkActive="active">
  <a class="nav-link active" routerLink="/">Home</a>
</li>
<li role="presentation" class="nav-item" routerLinkActive="active">
  <a class="nav-link" routerLink="/servers">Servers</a>
</li>
<li role="presentation" class="nav-item" routerLinkActive="active">
  <a class="nav-link" [routerLink]="['/users']">Users</a>
</li>

However, this will result in the 'Home' link always being 'active'...Not what we want
